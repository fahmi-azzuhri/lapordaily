# Lapor Daily
 ## Tech stack :
 - Vite ReactJS
 - TailwindCSS
 - Prisma
 - ExpressJS
 - MySQL
   
### Frontend
- cd frontend
- npm install
- npm run dev

### Backend
- cd backend
- npm install
- npx prisma migrate dev --name init
- node index.js

### DB
- run apache & mysql xampp control panel or any other web server
